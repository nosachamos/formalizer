# Third-party validators

Formalizer integrates with the awesome [validation](https://www.npmjs.com/package/validator) library.

Start by installing it:

```sh
yarn add validator
```

or

```sh
npm install validator --save
```

## Using third-party validators

Simply by installing it you can use all of their validators seamlessly:

```jsx
<input {...useInput('email', 'isEmail')} />
```

You can also pass options to third-party validators. To do so, create a custom validator that references the validator you want, and adds some options to it.

For example, to create a custom validator that errors when a field is empty, ignoring white spaces, we can build on the `isEmpty` third-party validator. From the validator's documentation:

> **`isEmpty(str [, options])`** - check if the string has a length of zero.
>
> `options` is an object which defaults to `{ ignore_whitespace: false }`.

We need to negate it, so it ensures the field is not empty. And we need to set the `ignore_whitespace` option to `true`.

```jsx
const notBlank = {
    validator: 'isEmpty' // <-- reference to te third party validator
    options: { ignore_whitespaces: true }, // options that will be applied to the validator
    negate: true // we want it to pass when the isEmpty validation fails, so negate that validator
}
```

Then you can use your custom validator normally:

```jsx
<input {...useInput('name', notBlank)} />
```
